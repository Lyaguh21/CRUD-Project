# CRUD-Project
## О проекте

Проект организован вокруг сущности `post` и реализует базовый CRUD через модальные формы: создание, просмотр, обновление и удаление. 

## Технологии:

- Vite 
- Mantine UI
- React
- Redux Toolkit
- TypeScript

## Структура проекта
- FSD


## Быстрый старт

1. Установите зависимости:

```bash
npm install
```

2. Запустить дев-сервер:

```bash
npm run dev
```
